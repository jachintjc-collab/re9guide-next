<h1>RE9 ティアリストハブ , 11 完全ランキング</h1>

<div>
<strong>日本語版について</strong> — このページは英語版の主要コンテンツへのナビゲーションを提供します。
個別のティアリスト記事は順次日本語化を進めています。完全な英語版は <a href="/tier-lists.html">こちら</a> でご覧いただけます。
</div>

<p>
バイオハザード レクイエム (RE9) のすべての主要ランキング可能要素を網羅する 11 のティアリスト。
武器、エンディング、ボス、章、お守り、アイテム、キャラクター、ロケーション、カットシーン、イースターエッグ、ベストモーメント。
合計 <strong>159 ランキング項目</strong>。
</p>
<blockquote>Mortal Edge made the knife-only run trivial. Worse, it made the regular run boring.</blockquote>

<h2>11 のティアリスト</h2>

<div>
<h3>1. <a href="/best-weapons-tier-list.html">RE9 武器ティアリスト (17 武器)</a></h3>
<p>マグナム + ショットガンが S ティア。すべての武器を戦闘実用性で S から D まで評価。v1.02 検証済み。</p>
</div>

<div>
<h3>2. <a href="/re9-best-endings-ranked-tier-list.html">RE9 エンディングランキング (8 エンディング)</a></h3>
<p>Grace + Leon キャンペーン全 8 エンディング。インサニティ専用シークレットエンディング含む。</p>
</div>

<div>
<h3>3. <a href="/re9-best-bosses-ranked-tier-list.html">RE9 ボスランキング (10 ボス)</a></h3>
<p>ヴィクター・ギデオン (#1) からドローン群 (#10) まで全ボスを戦闘デザインで評価。</p>
</div>

<div>
<h3>4. <a href="/re9-chapters-ranked-tier-list.html">RE9 チャプターランキング (8 章)</a></h3>
<p>ARK 施設 (#1) と RPD/孤児院 (#2) が S ティア。物語のペース、戦闘多様性で評価。</p>
</div>

<div>
<h3>5. <a href="/re9-all-charms-ranked-tier-list.html">RE9 お守りランキング (18 お守り)</a></h3>
<p>Crafter's Touch + Iron Skin が S ティア必須。プレイスタイル別最適ロードアウト含む。</p>
</div>

<div>
<h3>6. <a href="/re9-items-treasures-ranked-tier-list.html">RE9 アイテム / 宝物ランキング (18 アイテム)</a></h3>
<p>Eveline のロケット (#1) + マスターキー (#2)。プラチナトロフィー必須アイテム含む。</p>
</div>

<div>
<h3>7. <a href="/re9-characters-ranked-tier-list.html">RE9 キャラクターランキング (16 キャラクター)</a></h3>
<p>グレース・アシュクロフト (#1) + チャンク/マーカス (#2) + レオン (#3)。声優キャスト含む。</p>
</div>

<div>
<h3>8. <a href="/re9-locations-ranked-tier-list.html">RE9 ロケーションランキング (16 ロケーション)</a></h3>
<p>RPD メインホール (#1) + ARK 滅菌室 (#2) + Vivien 研究室 (#3)。フォトモード推奨スポット含む。</p>
</div>

<div>
<h3>9. <a href="/re9-cutscenes-ranked-tier-list.html">RE9 カットシーンランキング (16 カットシーン)</a></h3>
<p>Vivien 研究室公開 + Victor Gideon QTE + RPD メインホール入場が S ティア。再生時間情報付き。</p>
</div>

<div>
<h3>10. <a href="/re9-easter-eggs-ranked-tier-list.html">RE9 イースターエッグランキング (16 イースターエッグ)</a></h3>
<p>RPD RE2 リメイク複製 + Vivien 2009 研究ノート + ラスティスプーンが S ティア。RE シリーズ参照深度別。</p>
</div>

<div>
<h3>11. <a href="/re9-best-moments-ranked-tier-list.html">RE9 ベストモーメントランキング (16 モーメント)</a></h3>
<p>RPD メインホール入場 + Vivien 研究室日記 + Mr. X 追跡が S ティア。ゲームプレイ中心瞬間。</p>
</div>

<h2>方法論</h2>
<p>
各ティアリストは PS5 and PC での 6 周完全プレイのテスト結果に基づきます。
すべての評価は v1.02 パッチ後のバランス調整を反映。
評価は 5 次元加重スコアリング (合計 100%) を採用。
</p>
