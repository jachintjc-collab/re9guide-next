<h1>퍼펙트 Mr. X — 한 번도 맞지 않고 Mr. X 추격 회피</h1>
<p>챕터 6 Mr. X 추격 시퀀스 Mr. X 접촉 없이 완료. 챕터 다른 데미지 자격 박탈 안 됨; Mr. X 특정 접촉 데미지만 카운트.</p>

<blockquote>퍼펙트 Mr. X는 특정 라우팅 숙련 보상. 추격은 라우트 알면 어렵지 않음.</blockquote>

<h2>자격 박탈</h2><p>Mr. X 공격 동작 그레이스 타격. 순찰 중 직접 접촉 데미지.</p><h2>퍼펙트 Mr. X 루트</h2><p>보관실 출구 → 서쪽 복도(Mr. X 메인 로비) → 메인 로비 → 동쪽 계단. 뒤로 가지 마세요.</p><h2>세이브 관리</h2><p>챕터 시작에 저장. Mr. X 접촉 시 즉시 리로드.</p><h2>인새니티 퍼펙트 Mr. X</h2><p>Mr. X 더 빠른 이동 + 적극 순찰. 인새니티 스탠다드보다 훨씬 어려움.</p><h2>한국 퍼펙트 라우팅</h2><p>네이버 스피드런 커뮤니티 프레임 정밀 루트 타이밍.</p><h2>내 생각</h2><p>참여 플레이어 위한 5-10 시도. 인새니티 추구는 스탠다드 마스터 후.</p>
