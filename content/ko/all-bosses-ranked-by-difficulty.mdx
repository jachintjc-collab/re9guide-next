<h2 id="how">랭킹 기준</h2>
<p>각 보스는 다섯 가지 요소로 평가됩니다:</p>
<ul>
<li><strong>데미지 출력</strong>; 공격이 얼마나 강한지</li>
<li><strong>공격 패턴 복잡도</strong>. 몇 가지 동작이 있고 얼마나 읽기 쉬운지</li>
<li><strong>패리 난이도</strong> — 패리 윈도우가 얼마나 빡빡한지</li>
<li><strong>자원 소모</strong>; 탄약/회복 아이템을 얼마나 소비하는지</li>
<li><strong>인새니티 스케일링</strong>, 최고 난이도에서 얼마나 가혹해지는지</li>
</ul>
<p>목록은 가장 어려움(#1)부터 가장 쉬움(#8)까지. 순서가 중요합니다: 무엇이 올지 알면 올바른 전투에 자원을 아낄 수 있습니다.</p>

<h2 id="s">S 티어 — 잔혹한 전투</h2>

<h3>#1 빅터 기드온 박사 // ARK 시설 · 최종 ★★★★★</h3>
<p>레지던트 이블 레퀴엠의 가장 어려운 전투. 전기 공격, RPG-7, 촉수 슬램, 10초 보라 충전 윈도우의 2 페이즈 네메시스 클론. <strong>Mortal Edge Hatchet 필수.</strong> 이 전투 전에 탄약을 낭비하지 마세요.</p>
<p><em>CANON ENDING · 2-PHASE · LEON</em></p>

<h3>#2 더 커맨더 (HUNK 타입) // ARK 복도 ★★★★★</h3>
<p>RE4의 크라우저처럼 칼싸움. 빠른 공격에 읽기 쉬운 텔레그래프. 게임에서 가장 기술 집약적인 전투. 보상: <strong>Mortal Edge Hatchet</strong>, 빅터에 필수.</p>
<p><em>PARRY-FOCUSED · LEON</em></p>

<h3>#3 미스터 X. 슈퍼 타이런트 // RPD 외부 ★★★★★</h3>
<p>차 던지기와 도약 공격을 가진 상징적 보스. <strong>심장을 노리세요</strong> — 몸통 사격은 거의 데미지가 없습니다. 추격 시퀀스 먼저의 2 페이즈.</p>
<p><em>CHASE+FIGHT · LEON · HEART WEAKPOINT</em></p>

<h2 id="a">A 티어. 도전적인 전투</h2>

<h3>#4 타이탄 스피너 // 라쿤 시티 광장 ★★★★★</h3>
<p>거대한 회전 바이오웨폰. 회전 공격은 <strong>패리 불가</strong>, 엄폐물을 사용해야 합니다. 버스트 데미지가 일시적 스턴 윈도우를 트리거. 스턴용 레퀴엠 라운드를 아껴두세요.</p>
<p><em>COVER-BASED · LEON</em></p>

<h3>#5 플랜트 43 // ARK 입구 ★★★★★</h3>
<p>세 머리 식물 괴물. 옆 새싹을 먼저 사격한 다음 중앙 꽃이 입을 벌릴 때 사격. 마무리: 떨어진 트럭에서 폭발 캔 사격으로 즉시 시네마틱 처치.</p>
<p><em>CINEMATIC FINISHER · LEON</em></p>

<h2 id="b">B 티어 — 관리 가능한 전투</h2>

<h3>#6 청크 #2 (Timothy B. Jackson) // 케어 센터 다락방 ★★★★★</h3>
<p>두 번째 청크 조우. 전투 시작 시 시신 옆의 <strong>MSBG 500 샷건을 즉시 픽업</strong>. 노란 사다리로 경로를 강제하세요.</p>
<p><em>SHOTGUN PICKUP · LEON</em></p>

<h3>#7 청크 #1 (Thomas K. Jackson) // 케어 센터 동 윙 ★★★★★</h3>
<p>선택적이지만 권장. 문틀에 걸리는 느린 변이체. <strong>뒤에서 헤모리틱 인젝터 3개 = 즉사</strong>. Eye Spy 부적 드롭.</p>
<p><em>OPTIONAL · GRACE · CHARM REWARD</em></p>

<h3>#8 더 셰프 // 서 윙 주방 ★★★★★</h3>
<p>게임에서 가장 쉬운 "보스". 선택 조우. 무료 보급을 위한 팬트리 키 드롭. <strong>헤모리틱 인젝터 한 개로 즉사</strong>. 스피드런에서는 쉽게 건너뜁니다.</p>
<p><em>OPTIONAL · GRACE/LEON</em></p>

<h2 id="order">캠페인 진행 순서</h2>
<p>스토리 계획을 위해, 보스를 만나는 순서:</p>
<ol>
<li><strong>더 셰프</strong> (케어 센터 주방). 선택, 매우 초기</li>
<li><strong>청크 #1</strong> (동 윙) — 선택, Eye Spy 부적용 권장</li>
<li><strong>청크 #2</strong> (케어 센터 다락방), 필수, 중간 게임</li>
<li><strong>미스터 X / 슈퍼 타이런트</strong> (RPD 외부). 필수, 케어 센터 이후</li>
<li><strong>타이탄 스피너</strong> (라쿤 시티 광장) — 필수, 라쿤 시티 중간</li>
<li><strong>플랜트 43</strong> (ARK 입구); 필수, ARK 초기</li>
<li><strong>더 커맨더</strong> (ARK 복도); 필수, ARK 중간</li>
<li><strong>빅터 기드온</strong> (ARK 최종 챔버) — 최종 보스</li>
</ol>

<h2 id="prep">전투 전 체크리스트</h2>
<p>보편적인 보스 전 준비:</p>
<ul>
<li>전투 전에 수동 저장 (새 슬롯 사용)</li>
<li>가장 가까운 보급 상자에서 보디 아머 수리</li>
<li>응급 처치 스프레이 2개 이상 비축</li>
<li>컷씬을 트리거하기 전에 모든 무기 재장전</li>
<li>가장 강력한 근접 무기 장착 (이상적으로 Mortal Edge Hatchet)</li>
<li>인새니티의 경우 Eye Spy 부적 (무작위 치명상 회피) 장착 고려</li>
</ul>
