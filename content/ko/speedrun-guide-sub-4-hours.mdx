<h2>스피드런 셋업 요건</h2>

<div>
 💡 <strong>최고의 스피드런 조합:</strong> <strong>인새니티 + 무한 레퀴엠 장착</strong>으로 진행하세요. 이렇게 하면 스피드러너 트로피(5시간 미만)와 리마커블 에이전트 트로피(인새니티 클리어)를 한 번의 플레이로 동시에 달성합니다.
</div>

<h3>// 런 시작 전 해금</h3>

<ol>
 <li><strong>메인 스토리를 한 번 클리어</strong>. 인새니티 + 스페셜 콘텐츠 스토어 해금</li>
 <li><strong>무한 레퀴엠 구매</strong> 스페셜 콘텐츠 스토어에서 (~6,000 CP)</li>
 <li><strong>선택: 무한 RPG-7 구매</strong> (~8,000 CP). 빅터 보조용</li>
 <li><strong>이전 런의 Mortal Edge Hatchet 보유</strong> (NG+로 이월됨)</li>
</ol>

<h3>// 게임 내 설정</h3>

<ul>
 <li>옵션 메뉴에서 <strong>컷신 스킵</strong> ON 설정</li>
 <li><strong>자동 조준</strong>을 STRONG으로 설정 (귀중한 초 단위 절약)</li>
 <li>더 빠른 전환을 위해 <strong>조준 보조 토글</strong> 비활성화</li>
 <li><strong>퀵턴</strong>을 편한 버튼에 바인딩</li>
</ul>

<h2>필수 스킵 (총 ~45분 절약)</h2>

<div>
 <div><span>// 6분 절약</span>미스터 라쿤 25개 전부 스킵</div>
 <div>신경 쓰지 마세요. 스피드런 루트는 티어 3 무기 업그레이드가 필요 없습니다. 동선 바로 위에 있지 않은 메모리엄은 전부 스킵하세요.</div>
</div>

<div>
 <div><span>// 8분 절약</span>5개 금고 중 4개 스킵 (작업실 금고만 진행)</div>
 <div>작업실 금고(R60·L40·R80)는 라쿤 라운드업 지도를 줍니다 — 스피드런에는 쓸모없습니다. 모든 금고를 통째로 스킵하고; No Safe Is Safe 트로피는 포기하세요.</div>
</div>

<div>
 <div><span>// 12분 절약</span>선택 보스 스킵 (셰프 + 청크 #1)</div>
 <div>싸우지 마세요. 셰프와 청크 #1은 완전히 스킵 가능합니다. Eye Spy 참은 좋지만 무한 레퀴엠이 있으면 필수가 아닙니다.</div>
</div>

<div>
 <div><span>// 5분 절약</span>파일 75개 전부 스킵</div>
 <div>파일은 어떤 진행도 막지 않습니다. 모든 파일을 지나쳐 달리세요. "Case Closed!" 트로피는 수집런용이지 스피드런용이 아닙니다.</div>
</div>

<div>
 <div><span>// 8분 절약</span>정원 사이드 퀘스트 스킵</div>
 <div>씨앗 심기 = 선택. 그린 썸 트로피는 스피드러너와 함께 스택되지 않습니다.</div>
</div>

<div>
 <div><span>// 6분 절약</span>룰렛 휠 이스터에그 스킵</div>
 <div>RNG 의존적이고 골동 동전 1개만 절약합니다. 시간 투자 대비 가치가 없습니다.</div>
</div>

<h2>구간별 루트</h2>

<h3>// 렌우드 (목표: 12:00)</h3>

<p>선형 오프닝. 스킵 기회가 없습니다. 그냥 튜토리얼을 최대한 빠르게 진행하세요. 대사 빠르게 넘기고, 튜토리얼 적을 신속히 패리한 뒤, 복도를 따라가세요.</p>

<ul>
 <li>컷신을 즉시 스킵 (버튼 프롬프트)</li>
 <li>초반 탄약은 줍지 말 것, 나중에 충분히 얻습니다</li>
 <li>좀비를 지나쳐 달리고; 교전하지 마세요</li>
</ul>

<h3>// 케어 센터 그레이스 (목표: 55:00)</h3>

<p>가장 긴 구간. <strong>쿼츠 퍼즐 3개를 빠르게 모두 풀기</strong> — 필수입니다. 그 외에는 전부 스킵.</p>

<ol>
 <li><strong>태양 쿼츠</strong> (수석 연구원실): 별 · 태양 · 달 · 태양</li>
 <li><strong>달 쿼츠</strong> (의장실): 달 · 태양 · 별 · 달</li>
 <li><strong>별 쿼츠</strong> (VIP 스위트): 모래시계 해법 (보통 태양 · 별 · 달 · 별)</li>
</ol>

<p>중요: <strong>혈액 표본 1번 해결</strong> (변성), 용혈 주사기 확보를 위해. 나중에 2-3개가 필요합니다. 표본 2번과 3번은 스킵.</p>

<h3>// 케어 센터 레온 (목표: 25:00)</h3>

<p>다락방에서 MSBG 500을 즉시 픽업하세요. 무한 레퀴엠으로 다락방의 청크 #2를 원샷. 안뜰 출구로 곧장 달려가세요.</p>

<h3>// 라쿤시티 (목표: 60:00)</h3>

<p>여러 보스 전투:</p>
<ul>
 <li><strong>미스터 X 추격</strong>; 싸우지 말고 루트만 달리세요 (미스터 X 공략 참조)</li>
 <li><strong>슈퍼 타이런트</strong> — 무한 레퀴엠으로 심장 사격. ~90초</li>
 <li><strong>타이탄 스피너</strong>; 무한 레퀴엠으로 엄폐-점사. ~3분</li>
</ul>

<p>중요: 건샵 켄도의 저장고에서 시간을 낭비하지 마세요, 무한 레퀴엠만 있으면 충분합니다.</p>

<h3>// ARK 시설 (목표: 55:00)</h3>

<p>플랜트 43 → 커맨더 → 복도 → 빅터:</p>
<ul>
 <li><strong>플랜트 43</strong> — 캐니스터 트럭 폭발 피니셔 노리기 (페이즈 2 스킵)</li>
 <li><strong>커맨더</strong>; Mortal Edge 패리 사용; 인새니티 + 무한 레퀴엠으로 ~3분</li>
 <li><strong>ARK 복도</strong>, 순수 달리기 통과. 전투 불필요.</li>
</ul>

<h3>// 빅터 기든 (목표: 15:00)</h3>

<p>최종 보스. 무한 레퀴엠으로:</p>
<ol>
 <li>페이즈 1: 보라색 차지 결정 사격 → 2분</li>
 <li>페이즈 2: 머리의 농포 파괴 → 코어 → 5-8분</li>
 <li>QTE 피니셔 프롬프트</li>
</ol>

<h2>4시간 미만 런용 로드아웃</h2>

<div>
 <strong>최적 로드아웃:</strong><br>
 주무기: <strong>무한 레퀴엠 리볼버</strong> (대부분의 적을 원샷)<br>
 보조: <strong>무한 RPG-7</strong> (긴급 보스 데미지)<br>
 근접: <strong>Mortal Edge Hatchet</strong> (패리 전용, 절대 공격하지 말 것)<br>
 참: <strong>Eye Spy Charm</strong> 획득 가능하면 (랜덤 치명타 세이브)
</div>

<h2>세이브 전략</h2>

<p>스피드런은 <strong>세이브가 적을수록</strong> 유리합니다:</p>

<ul>
 <li>주요 보스 전(미스터 X, 플랜트 43, 커맨더, 빅터) 직전에만 세이브</li>
 <li>RNG 요소 전(모래시계 퍼즐)에 세이브</li>
 <li>탐색 중에는 세이브하지 말 것 — 각 세이브당 ~5초 손실</li>
 <li>같은 슬롯 사용; 멀티 슬롯 세이브는 시간 낭비</li>
</ul>

<h2>흔한 시간 손실 실수</h2>

<ul>
 <li><strong>파일을 읽으려고 멈추기</strong>. 절대 금지. 전부 스킵 가능합니다.</li>
 <li><strong>선택 좀비와 교전</strong> — 지나쳐 달리세요. 한 발만 쏴도 2-3초 낭비.</li>
 <li><strong>불필요한 탄약 줍기</strong>; 무한 레퀴엠으로 의미 없습니다.</li>
 <li><strong>실수로 사이드 보스 트리거</strong>. 청크 #1과 셰프의 트리거 지점을 알아두세요.</li>
 <li><strong>너무 자주 세이브</strong> — 세이브 1회당 4-6초 추가.</li>
 <li><strong>태양 퍼즐을 틀리게 풀기</strong>, 답을 외우고, 단서를 다시 읽지 마세요.</li>
</ul>

 <li><strong>평균 인새니티 스피드런:</strong> 3:40-4:00 (무한 탄약 사용)</li>
