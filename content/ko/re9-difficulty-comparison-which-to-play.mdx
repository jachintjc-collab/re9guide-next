<h2 id="tldr">TL;DR; 어떤 난이도를 플레이해야 하나?</h2>
<ul>
 <li><strong>첫 RE 플레이어:</strong> Casual. RE9은 프랜차이즈의 가장 어려운 작품; Casual은 스토리를 사소화하지 않으면서 진정으로 캐주얼.</li>
 <li><strong>RE2 Remake / RE4 Remake 익숙한 사람:</strong> Standard. 이것이 의도된 경험; 스토리 긴장과 전투 모두 옳게 느껴짐.</li>
 <li><strong>진짜 도전 원하지만 고문은 아닌:</strong> Hardcore. 빠듯한 탄약, 제한된 저장, 그러나 Insanity 전용 원샷 공격은 없음.</li>
 <li><strong>트로피 헌터 / 컴플리셔니스트:</strong> 캐리오버 장비로 Insanity NG+. 3개 트로피 + Survivor 골드 트로피에 필요.</li>
 <li><strong>트로피에 신경 쓰지 않지만 "최고 버전"을 원한다면:</strong> Standard 먼저, 그 후 NG+ Hardcore. 재플레이당 8+ 시간 여유가 없다면 Insanity 건너뛰기.</li>
</ul>

<h2 id="boss-changes">보스 공격 패턴 변경 (Insanity 전용 기술)</h2>
<p>여러 보스가 Insanity에서 낮은 난이도에 존재하지 않는 완전히 새로운 공격 획득:</p>

<ul>
 <li><strong><a href="how-to-beat-victor-gideon.html">빅터 기드온</a></strong> — Insanity는 페이즈 4 "Mortal Edge 스팸" 시퀀스 추가: 4초 안에 6개의 연속 Mortal Edge 공격. 완벽한 패리 타이밍 또는 즉사 필요. 이 페이즈는 Hardcore 이하에 존재하지 않음.</li>

 <li><strong><a href="how-to-beat-the-commander.html">사령관 (The Commander)</a></strong>; Insanity는 플레이어 HP가 30% 미만일 때 즉사 "명령 오버라이드" 공격 추가. Eye Spy Charm 카운터. 부적이 이 공격에서 당신을 구할 수 없음. 회피 롤 캔슬을 통해서만 회피 가능.</li>

 <li><strong><a href="how-to-beat-titan-spinner.html">타이탄 스피너</a></strong> — Insanity는 전체 4페이즈 추가: 메인 본체 격파 후, 5배 피해 공격을 하는 노출된 코어가 등장. 총 전투 길이가 두 배.</li>

 <li><strong><a href="how-to-beat-chunk.html">청크 (Chunk)</a></strong>; Insanity는 최종 QTE 프롬프트를 5초에서 3초로 변경. 놓치면 "샘플 파괴"가 기본값 (낮은 난이도와 동일), 그러나 시간 압박으로 많은 플레이어가 실수로 프롬프트를 완전히 실패.</li>

 <li><strong><a href="how-to-beat-mr-x-super-tyrant.html">Mr. X / 슈퍼 타이런트</a></strong>. Insanity는 Mr. X를 효과적으로 스턴 불가능하게 만듦. Requiem Revolver 헤드샷도 0.3초만 비틀거리게 함 (Hardcore의 2초 vs).</li>
</ul>

<h2 id="path">권장 난이도 진행 경로</h2>

<p><strong>캐주얼 접근 (가장 낮은 위험, 가장 느림):</strong> Casual → Standard → Hardcore → Insanity. 4개 별도 플레이. 총 ~60시간. 첫 RE 플레이어에게 최고.</p>

<p><strong>최적 접근 (권장):</strong> Standard → NG+ Hardcore → NG+ Insanity. 3개 플레이. 총 ~30시간. Standard에서 의도된 난이도 곡선을 경험할 수 있고 Hardcore가 Insanity에 대한 의미 있는 준비이므로 Casual 건너뛰기.</p>

<p><strong>트로피 러시 접근:</strong> Standard → NG+ Insanity (Hardcore 건너뛰기!). 2개 플레이. ~24시간. 복귀하는 RE 베테랑인 경우에만 실행 가능. Insanity에 무한 Requiem Revolver이 없어 잔인해짐. 이것이 당신에게 실행 가능한지는 <a href="re9-new-game-plus-complete-guide.html">우리 NG+ 완전 가이드</a> 참조.</p>

<p><strong>스피드런 접근:</strong> Standard (최적 루트로) → NG+ Hardcore (무한 Requiem Revolver 확인). 총 ~14시간. Insanity는 포함하지 않지만 콘텐츠의 90%를 해금.</p>

<h2 id="worth">Insanity 가치 있나? (솔직한 판정)</h2>

<p><strong>트로피 헌터:</strong> 필수. 3개 트로피가 Insanity 전용이고 플래티넘에 필요.</p>

<p><strong>Lore 애호가:</strong> 강력 권장. Insanity는 전체 Chunk 코덱스 (Marcus Halverson 배경), Eveline 배양조 등장, 47명 개발자 감사 카드 해금. 놓칠 것은 <a href="chunk-origin-mystery-explained.html">Chunk 기원 미스터리</a> 참조.</p>

<p><strong>순수 게임플레이 즐거움:</strong> 상황에 따라. Insanity의 보스 전투 변경은 기계적으로 흥미로움 — 빅터 기드온의 페이즈 4와 타이탄 스피너의 확장된 전투는 진정으로 Standard 버전보다 더 잘 설계됨. 그러나 지속적인 탄약 부족은 갈리는 느낌일 수 있음.</p>

<p><strong>솔직한 "Insanity 건너뛰기" 케이스:</strong> Hardcore를 끝내고 "이 게임에서 원하는 것을 얻었다"고 느꼈다면, Insanity 건너뛰기 합리적. 3개 트로피와 일부 코덱스 콘텐츠를 놓침. 8-12시간 재플레이는 모두에게 맞지 않음.</p>

<p><strong>솔직한 "Insanity 플레이" 케이스:</strong> Hardcore의 긴장을 즐겼고 프랜차이즈가 제공한 가장 깊은 RE 전투 경험을 원한다면, Insanity는 프랜차이즈의 기술적 정점. 빅터 기드온 페이즈 4 전투만으로도 RE 역사상 최고의 단일 보스 인카운터로 널리 인용됨.</p>

<h2 id="troubleshoot">"X 난이도에서 막혔어". 결정 트리</h2>

<p><strong>Standard의 Chunk 전투에서 막혔다?</strong> → 캠페인 완료를 위해 Casual로 내림. 선택적 자원 수집을 건너뛰면 Standard 어려움. 나쁘게 느끼지 마세요; ~30%의 플레이어가 여기서 다운시프트.</p>

<p><strong>Hardcore의 빅터 기드온에서 막혔다?</strong> → <a href="re9-eye-spy-charm-explained.html">Eye Spy Charm</a> 장착 (PT1에서 이전) + 진입 전 응급치료 스프레이 4개 비축. 여전히 막혔다면, 전투 중간에 Standard로 다운시프트; 난이도 경계까지 난이도 변경 가능.</p>

<p><strong>Insanity의 어떤 보스에서 막혔다?</strong> → NG+ Hardcore의 캐리오버 Requiem Revolver 사용. 무한 Requiem Revolver이 없다면, 권장 경로를 건너뜀. 먼저 NG+ Hardcore로 돌아가세요.</p>

<p><strong>Insanity의 좀비 떼 (보스 아님)에서 막혔다?</strong> → 전투 중심에서 스텔스 중심 플레이스타일로 전환. 인카운터의 60% 웅크려 걷기. 보스에만 총알 절약.</p>

<p><strong>Insanity의 탄약 부족에서 막혔다?</strong> → 제작을 주요 수입원으로 만들기. 근접 마무리를 위해 칼 8개 이상 휴대 (칼 킬은 탄약을 소비하지 않고 +1 제작 수율 제공).</p>

<h2 id="faq">FAQ — RE9 난이도 메커닉</h2>

<p><strong>Q: 플레이 중간에 난이도를 변경할 수 있나요?</strong><br>
A: 낮추는 것만. Insanity → Hardcore는 모든 저장 지점에서 허용. 올리는 것 (Standard → Hardcore)은 난이도 시작 후 잠김.</p>

<p><strong>Q: 난이도가 엔딩에 영향을 미치나요?</strong><br>
A: 대부분 아니요; 모든 8개 엔딩은 난이도 전반에서 접근 가능 EXCEPT: Grace Insanity와 Leon Insanity 엔딩은 Insanity 난이도 클리어 필요. 전체 조건은 <a href="re9-all-endings-explained.html">엔딩 가이드</a> 참조.</p>

<p><strong>Q: 트로피가 NG+ 난이도 변경에 걸쳐 이전되나요?</strong><br>
A: 네. 획득한 트로피 지속. PT1에서 "Hardcore Complete"를 얻었다면, Casual에서 재플레이해도 당신 것.</p>

<p><strong>Q: Casual은 "이지 모드 부끄러움"으로 간주되나요?</strong><br>
A: 아니요. 캡콤은 명시적으로 내러티브 중심 플레이어를 위해 Casual을 설계. 스토리 경험은 동일; 전투에 더 적은 시간 소비. 많은 평론가가 리뷰 플레이를 위해 Casual을 플레이.</p>

<p><strong>Q: 낮은 난이도에서 이긴 보스에 대해 난이도가 어떻게 영향을 미치나요?</strong><br>
A: 영향 없음. 각 플레이는 새로운 시도. Standard에서 빅터 기드온을 100% 했다 해도, Insanity에서는 새로 그와 싸움.</p>

<p><strong>Q: 실제로 Insanity를 끝낸 플레이어의 비율은?</strong><br>
A: PS5 트로피 데이터 (2026년 5월): 소유자의 4.2%가 "Insanity Complete" 트로피 보유. ~12%가 Hardcore. ~78%가 Standard 이상. Casual 전용은 ~6%.</p>

<p><strong>Q: 트로피를 위해 Insanity에서 무한 Requiem Revolver을 사용할 수 있나요?</strong><br>
A: 네. 무한 탄약은 RE9에서 트로피를 무효화하지 않음 (일부 이전 RE 작품과 달리). 더 안전한 Insanity 완료를 위해 자유롭게 사용.</p>

<p><strong>Q: Eye Spy Charm은 Insanity에서 작동하나요?</strong><br>
A: 네, 단 하나의 주의사항: 사령관의 Insanity 전용 "명령 오버라이드" 즉사 공격은 Eye Spy Charm을 우회. 다른 모든 죽음 방지는 정상적으로 작동.</p>
