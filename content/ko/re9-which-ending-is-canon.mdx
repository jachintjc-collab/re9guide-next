<p><strong>짧은 답:</strong> <strong>Grace True 엔딩</strong>이 공식 정사. <strong>Leon 인새니티 시크릿 엔딩</strong>이 대체 정사 (RE10 설정). 다른 모든 엔딩은 비정사. 디렉터 Nakanishi가 2026년 5월 Famitsu 인터뷰에서 확인.</p>
<h2>정사 엔딩: Grace True 엔딩</h2>
<p>달성 방법: 표준 난이도 이상에서 나이프 패리 (0.4초 윈도우)로 Victor Gideon의 QTE 피날레 생존.</p>
<p><strong>일어나는 일:</strong> Grace가 ARK 시설 탈출. Mia Winters가 EW-2017 헬리콥터로 도착. Vivien-Eveline 연결 시네마틱 재생. Grace가 Eveline's Locket을 들고 서울로 돌아옴. RE10 주인공으로 Grace 설정.</p>

<h2>대체 정사: Leon 인새니티 시크릿 엔딩</h2>
<p>달성 방법: Leon으로 NG+ 인새니티 완료 + 의도적으로 Victor Gideon QTE 패리 놓치기.</p>
<p><strong>일어나는 일:</strong> Leon이 심한 부상으로 생존. Ada Wong 비콘 재생, 그녀가 그를 추적 중. DLC 그리고/또는 RE10 재회 설정.</p>

<h2>비정사 엔딩</h2>
<ul>
 <li>Grace 배드 엔딩, Grace가 캐주얼에서 QTE 실패</li>
 <li>Leon True 엔딩 — 성공하지만 Ada 비콘 트리거 안 함</li>
 <li>배드 엔딩 (Grace + Leon); 캐릭터 사망으로 스토리 끝</li>
 <li>인새니티 표준 엔딩; 표준과 같은 콘텐츠 + 보너스 코덱스</li>
</ul>

<h2>정사 확인 방법: 공식 출처</h2>
<ul>
 <li><strong>Famitsu 인터뷰 (2026년 5월):</strong> 디렉터 Nakanishi가 Grace True 엔딩이 정사라고 명시</li>
 <li><strong>RE9 공식 아트북:</strong> Grace True 엔딩 아트만 포함</li>
 <li><strong>캡콤 커뮤니티 포스트:</strong> "Grace의 여정이 계속됨" 참조 — 생존 시사</li>
</ul>
