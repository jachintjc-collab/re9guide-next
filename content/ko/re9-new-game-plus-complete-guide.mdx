<h2 id="mechanic">NG+ 작동 방식 (규칙)</h2>
<p>RE9의 NG+ 시스템은 몇 가지 특정 특이점이 있는 현대 캡콤 표준을 따릅니다:</p>
<ol>
 <li><strong>캐릭터별 세이브:</strong> 각 캐릭터 (Grace와 Leon)는 독립적인 NG+ 진행을 가집니다. Grace의 캠페인을 클리어하면 Grace NG+만 해금. Leon도 마찬가지.</li>
 <li><strong>NG+ 진입 시 난이도 선택:</strong> NG+로 들어갈 때 난이도를 변경할 수 있습니다 (예: Standard에서 완료, Insanity에서 NG+ 시작).</li>
 <li><strong>아이템 캐리오버:</strong> 대부분의 비-퀘스트 아이템이 이전됩니다. 퀘스트 중요 아이템 (키 카드, 스토리 아이템)은 "누락"으로 재설정; 다시 획득해야 합니다.</li>
 <li><strong>맵 상태 재설정:</strong> 모든 퍼즐 상태, 잠긴 문, 영역 진행 재설정. 캐리오버 장비로 같은 물리적 캠페인을 다시 플레이.</li>
 <li><strong>엔딩 플래그 지속:</strong> 난이도 변경에도 게임은 해금한 엔딩을 기억합니다. <a href="/ko/re9-all-endings-explained/">엔딩 가이드 참조</a> 전체 엔딩 해금 매트릭스.</li>
</ol>
<p>이 시스템은 RE2 Remake (NG+ 캐리오버 제한)보다 관대하고 RE4 Remake (보너스 모드를 통해 캐릭터 간 이전 허용)보다 엄격합니다.</p>

 위치: NG+ Insanity에서 Chunk를 격파한 후 그의 시체. Grace의 Secret Ending에 필요. 파일이 포함하는 것은 <a href="/ko/chunk-origin-mystery-explained/">Chunk 기원 미스터리</a> 참조.</li>

 위치: ARK 시설 서버실 (Leon 캠페인). Leon의 Secret Ending에 필요. 전체 공개는 <a href="/ko/ada-wong-re9-reversal-arc/">Ada Wong 반전 아크</a> 참조.</li>

 <li><strong>봉인된 아카이브실 키</strong> (NG+ 전용)<br>

 <li><strong>실험실 기술자의 일지</strong> (NG+ Hardcore+ 전용)<br>

 <li><strong>Eveline 참조 사진</strong> (NG+ Insanity 전용)<br>
 위치: 멸균실의 파괴 가능한 벽 뒤. Grace의 True Ending 미드 크레딧 장면에서 추가 대사 옵션 트리거.</li>

<h2 id="farm-order">100%를 위한 최적 NG+ 파밍 순서</h2>
<p>8개 엔딩 모두 + 모든 NG+ 치트 + 모든 의상 + 플래티넘 트로피를 최소 플레이타임으로 해금하려면, 이 정확한 순서를 따르세요:</p>

<ol>
 <li><strong>플레이 1. Grace Casual:</strong> 최적화가 아닌 게임 학습 목표. 모든 25 Mr. Raccoon 수집 (업그레이드 티어 이전). "샘플 파괴" 선택. (~14시간)</li>

 <li><strong>플레이 2 — Leon Casual:</strong> 같은 접근법. "샘플 보존" 선택 → Leon True Ending 해금 (Grace True 플래그가 이전되기 때문). (~13시간)</li>

 <li><strong>플레이 3; Grace NG+ Hardcore:</strong> 이제 풀 장비 + 탄약. Hardcore 클리어로 무한 Requiem Revolver + Crafter's Charm 해금. Grace Bad Ending을 위해 "샘플 보존" 선택. (~8시간)</li>

 <li><strong>플레이 4, Leon NG+ Standard:</strong> 서버실에서 Ada의 비콘 픽업. "샘플 보존" 선택 → Leon Secret Ending. (~6시간)</li>

 <li><strong>플레이 5 — Grace NG+ Insanity:</strong> 캐리오버 장비 + 무한 Requiem Revolver 사용. Chunk의 암호화된 파일 픽업. "샘플 파괴" 선택 → Grace Secret Ending. 격파 후, 세이브 리로드, NG+ 캐리 무기 사용 없이 Chunk 재플레이 → Grace Insanity Ending. (~10시간)</li>

 <li><strong>플레이 6; Leon NG+ Insanity:</strong> 컨테이너 손상 플래그를 트리거하기 위해 탈출 시퀀스 동안 한 번 맞기. "샘플 보존" 선택 → Leon Insanity Ending. 또한 무한 로켓 + Defender's Charm 해금. (~8시간)</li>
</ol>

<p><strong>100%를 위한 총 시간:</strong> ~59시간. 각 플레이 내 정확한 트로피 타이밍은 <a href="/ko/all-49-trophies-platinum-guide/">플래티넘 트로피 가이드와 결합</a>.</p>

<h2 id="traps">흔한 NG+ 함정 (이 실수들을 하지 마세요)</h2>

<ol>
 <li><strong>Hardcore를 건너뛰고 NG+를 Insanity에서 시작하지 마세요.</strong> Insanity는 무한 Requiem Revolver (Hardcore 완료 후에만 해금)을 가정합니다. Standard → Insanity로 가면 Plant 43이 거의 불가능합니다.</li>

 <li><strong>NG+ 전에 옛 무기를 분해하지 마세요.</strong> 일부 플레이어는 캠페인 종료 전에 무기를 판매하거나 폐기, 이것들은 NG+에서 돌아오지 않습니다. 모든 것을 유지하세요.</li>

 <li><strong>Leon Secret을 원한다면 Ada 비콘을 건너뛰지 마세요.</strong> 비콘은 리스폰되지 않습니다 — NG+ Standard/Hardcore 플레이에서 놓치면 다시 시도하기 위해 또 다른 완전한 NG+ 플레이가 필요합니다.</li>

 <li><strong>Insanity 외부에서 Chunk의 암호화된 파일을 픽업하지 마세요.</strong> 파일은 NG+ Insanity에서만 스폰. 낮은 난이도에서 시도하면 시간 낭비.</li>

 <li><strong>Grace Insanity Ending 시도에서 NG+ 캐리 무기를 사용하지 마세요.</strong> 게임은 캐리오버 무기 사용 여부를 추적. 한 개라도 사용하면 "The Cycle Continues" 트로피 자격 박탈.</li>

 <li><strong>Chunk-R2-준비 세이브를 덮어쓰지 마세요.</strong> Chunk 라운드 2 전 USB/클라우드에 수동 세이브 백업으로 다른 완전한 플레이 없이 엔딩 재시도 가능.</li>

 <li><strong>초기 NG+에서 새 무기를 구매하지 마세요.</strong> 이미 모든 것이 이전되어 있습니다. 새 무기 구매는 업그레이드에 가야 할 크레딧을 낭비.</li>
</ol>

<h2 id="tips">상위 10가지 NG+ 효율성 팁</h2>

<ol>
 <li><strong>첫 NG+ 스피드런 루트:</strong> 선택적 콘텐츠 건너뛰기; 메인 목표 돌진. NG+ 스케일링은 Standard 플레이를 6-7시간에 끝낼 수 있게 합니다.</li>
 <li><strong>Insanity 필수품 사전 제작:</strong> NG+ Insanity의 챕터 6 전에 Requiem Revolver 99발, 응급치료 스프레이 30개.</li>
 <li><strong><a href="/ko/re9-eye-spy-charm-explained/">Eye Spy Charm</a> 즉시 장착:</strong> PT1에서 이전. 자동 부활이 Insanity 플레이를 구합니다.</li>
 <li><strong>모든 보스 전투 전에 저장:</strong> NG+ 자동 저장은 더 나은 엔딩에서 잠금 가능.</li>
 <li><strong>Hardcore를 스위트 스폿으로 사용:</strong> Insanity의 잔인한 스케일링 없이 무한 Requiem Revolver 해금.</li>
 <li><strong>Grace와 Leon을 병렬로 실행:</strong> 그들의 엔딩이 상호 참조. Leon을 시작하기 전에 모든 4개 Grace 엔딩을 완료하지 마세요.</li>
 <li><strong><a href="/ko/all-quartz-puzzle-solutions/">퍼즐 해법</a>을 북마크:</strong> NG+에서도 퍼즐 상태 재설정. 해법을 미리 알면 플레이당 30분 이상 절약.</li>
 <li><strong>엔딩 전용 플레이에 Casual 사용:</strong> True→Bad 엔딩 플래그만 뒤집어야 한다면, Casual은 마지막 2개 챕터를 90분에 블리츠 가능.</li>
 <li><strong>노트북에 엔딩 해금 추적:</strong> 게임 내 엔딩 카운터는 0/8 → 8/8을 보여주지만 어떤 것인지 나열 안 함. 수동 추적으로 반복 방지.</li>
 <li><strong>설정에서 코덱스 자동 팝업 활성화:</strong> NG+ Insanity는 코덱스 항목 해금, 자동 팝업으로 <a href="/ko/chunk-origin-mystery-explained/">Chunk의 전체 기원 공개</a>를 놓치지 않음.</li>
</ol>

<h2 id="faq">FAQ, NG+ 메커닉</h2>

<p><strong>Q: Casual NG+가 Insanity NG+가 해금하는 모든 것을 해금하나요?</strong><br>
A: 아니요. 각 난이도는 자체 완료 보상이 있습니다. 모든 NG+ 콘텐츠를 해금하려면 각 난이도 (Casual → Standard → Hardcore → Insanity)에서 최소 한 번씩 클리어해야 합니다.</p>

<p><strong>Q: NG+ 중간에 난이도를 변경할 수 있나요?</strong><br>
A: 네, 하지만 낮추는 것만. Insanity → Hardcore로 중간 다운그레이드 가능하지만, 올리는 것 (Standard → Insanity)은 플레이 중간에 허용되지 않습니다.</p>

<p><strong>Q: Grace의 엔딩 카운트가 Leon의 NG+로 이전되나요?</strong><br>
A: 엔딩 플래그만 이전, 무기/아이템은 아님. 이것이 Leon True Ending이 세이브에 "Grace True 플래그 = true"를 필요로 하는 이유입니다.</p>

<p><strong>Q: 해금한 엔딩을 어떻게 알 수 있나요?</strong><br>
A: 메인 메뉴 → 엔딩 갤러리 (첫 엔딩 후 해금). 잠긴 것은 회색으로 표시된 8개 엔딩 모두의 2×4 그리드 보여줌.</p>

<p><strong>Q: Insanity를 한 번도 플레이하지 않고 8개 엔딩 모두 얻을 수 있나요?</strong><br>
A: 아니요. 3개 엔딩은 Insanity 필요: Grace Insanity, Grace Secret (Chunk 파일에 NG+ Insanity 필요), Leon Insanity. 다른 5개 엔딩은 낮은 난이도에서 접근 가능.</p>

<p><strong>Q: 플래티넘 트로피는 NG+가 필요한가요?</strong><br>
A: 네 — 여러 트로피가 NG+ 독점 필요 (예: Grace Secret의 "What Was Hidden", Leon Insanity의 "Quarantined"). NG+는 플래티넘에 필수.</p>

<p><strong>Q: NG+를 무한 탄약 치트와 결합할 수 있나요?</strong><br>
A: 네. 무한 탄약은 트로피를 비활성화하지 않음 (일부 이전 RE 게임과 달리). NG+ Insanity에서 안전한 트로피 사냥을 위해 자유롭게 사용.</p>

<p><strong>Q: NG+에 "영원히 잃어버리는" 아이템이 있나요?</strong><br>
A: 하나, ARK Lower Lab 환기구의 실험실 기술자의 일지. Lower Lab 영역을 떠나기 전에 픽업하지 않으면 그 플레이에서 다시 들어가서 잡을 수 없습니다. 보스 전투 전에 항상 픽업하세요.</p>
