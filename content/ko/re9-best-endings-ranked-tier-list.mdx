<h2 id="methodology">랭킹 방법론</h2>
<p>각 엔딩은 5개 차원으로 점수 매김, 합성이 티어 배치 결정:</p>

<div>
 <table>
 <tr><th>차원</th><th>측정 대상</th><th>가중치</th></tr>
 <tr><td><strong>내러티브 보답</strong></td><td>엔딩이 스토리 스레드를 만족스럽게 해결?</td><td>30%</td></tr>
 <tr><td><strong>캐릭터 아크 완성</strong></td><td>주인공이 의미 있게 성장하거나 변화?</td><td>25%</td></tr>
 <tr><td><strong>프랜차이즈 중요성</strong></td><td>엔딩이 RE10/미래 작품에 중요?</td><td>20%</td></tr>
 <tr><td><strong>노력-보상 비율</strong></td><td>해금 난이도가 보답에 비례?</td><td>15%</td></tr>
 <tr><td><strong>재플레이/논의 가치</strong></td><td>엔딩이 커뮤니티 참여 생성?</td><td>10%</td></tr>
 </table>

<p>이것은 "스포일러 없는 엔딩 순서" 가이드가 아닙니다 — 그건 다른 문서. 이것은 각 엔딩이 <em>당신의 시간을 가질 가치가 있는지</em>에 기반한 티어 리스트 랭킹. 두 True 엔딩이 지배하는 것은 둘 다 정사적이고 감정적으로 풍부하기 때문. Bad 엔딩은 의미 있는 내러티브가 아닌 "잘못된 선택을 하면 어떻게 되나"로 주로 존재하므로 마지막에 마무리.</p>

<h2 id="tldr">TL;DR, 어떤 것을 먼저</h2>
<ul>
 <li><strong>먼저 플레이:</strong> Grace True → Leon True (둘 다 S-티어, 서로 위에 구축)</li>
 <li><strong>두 번째 플레이를 위해 저장:</strong> Grace Secret + Leon Secret (NG+ 필요)</li>
 <li><strong>트로피 헌터:</strong> 플래티넘을 위해 8개 엔딩 모두 필요. 최적 파밍 순서는 <a href="re9-new-game-plus-complete-guide.html">NG+ 가이드</a> 사용.</li>
 <li><strong>2개만 시간 있다면:</strong> Grace True + Grace Secret. Grace 아크가 RE9의 감정적 중심.</li>
 <li><strong>4개만 시간 있다면:</strong> Leon True + Grace Insanity 추가로 3 주인공 모두의 주요 해결 커버.</li>
</ul>

<h2 id="rank1">#1: Grace True 엔딩; "A Quiet Life" (S-티어)</h2>
<p><strong>#1인 이유:</strong> 이것은 RE9의 감정적 중심. Grace가 헬리콥터에서 기다리는 Mia Winters 발견, ARK 시설에서 떠나고, 복수보다 치유 선택. Grace가 어머니 목걸이를 해변에 두고 바다로 걸어들어가는 미드 크레딧 장면은 20+ 년 만의 프랜차이즈에서 가장 시적인 순간.</p>

<p><strong>해금 노력:</strong> ★ 쉬움 (어떤 난이도 + "샘플 파괴")</p>
<p><strong>최적 대상:</strong> 첫 플레이어, lore 팬, 캡콤 정사 추종자 (이것이 의도된 엔딩)</p>
<p><strong>트로피:</strong> "A Quiet Life" (실버)</p>

<details>
<summary>▶ 공개: 왜 이 엔딩이 그렇게 강하게 다가오나</summary>
<div>
<p>Grace의 True 엔딩이 성공하는 이유는 카타르시스를 얻기 때문. 트라우마에 추진된 14시간의 조사 후, Grace는 마침내 <em>멈출</em> 기회 얻음. Mia Winters의 헬리콥터 등장은 프랜차이즈의 가장 큰 감정적 다리 — RE7 (2017)에서 RE9 (2026)으로 9년 루프 닫음. 디렉터 Koshi Nakanishi는 Famitsu에서 이것이 RE7을 끝낸 후 계획한 순간이라고 확인. <a href="mia-winters-re9-re7-connection.html">Mia 연결 심층 분석</a> 참조.</p>
</div>
</details>

<h2 id="rank2">#2: Leon True 엔딩. "Necessary Evil" (S-티어)</h2>
<p><strong>#2인 이유:</strong> Leon이 RNA 샘플을 Chris Redfield에게 전달, 공식적으로 BSAA에 합류, 28년 단독 요원 상태 종료. "커피?" "그래. 커피." 교환은 프랜차이즈의 가장 조용하게 강력한 캐릭터 비트 중 하나. Grace True 해금이 먼저 필요하므로 Grace True보다 약간 아래.</p>

<p><strong>해금 노력:</strong> ★★ 중간 (먼저 Grace True 세이브 필요, 그 후 어떤 난이도 + "샘플 보존")</p>
<p><strong>최적 대상:</strong> 오랜 Leon 팬, BSAA lore 추종자</p>
<p><strong>트로피:</strong> "Necessary Evil" (실버)</p>

<details>
<summary>▶ 공개: BSAA 전환의 장기적 중요성</summary>
<div>
<p>Leon이 BSAA에 합류하는 것은 프랜차이즈의 25년 평행 요원 설정 (Leon vs Chris) 붕괴. 처음으로 모든 주요 영웅이 한 제도적 우산 아래. 이것은 2004년 BSAA 설립 이후 캡콤의 가장 큰 조직적 재설정. <a href="bsaa-re9-new-direction.html">BSAA 새 방향 분석</a> 참조.</p>
</div>
</details>

<h2 id="rank3">#3: Grace Secret 엔딩, "What Was Hidden" (A-티어)</h2>
<p><strong>#3인 이유:</strong> Eveline의 불가리아 감시 사진은 게임 전체에서 가장 중요한 단일 플롯 공개. Eveline이 RE7에서 살아남았음 확인, RE10 설정, Grace 아크를 진행 중인 것으로 재구성. NG+ 인새니티 + 특정 아이템 픽업 필요하므로 True 엔딩보다 약간 아래.</p>

<p><strong>해금 노력:</strong> ★★★★ 어려움 (NG+ 인새니티 + Chunk 암호화 파일 + "샘플 파괴")</p>
<p><strong>최적 대상:</strong> lore 애호가, RE10 추측 팬</p>
<p><strong>트로피:</strong> "What Was Hidden" (골드)</p>

<details>
<summary>▶ 공개: 불가리아 사진 디코드</summary>
<div>
<p>Varna, 불가리아 감시 이미지 (날짜 20.04.2026 — Grace 캠페인 시작 정확히 14일 전)는 Eveline이 살아있고 어린이 형태임 확인. Grace의 "엄마가 살아있어" 대사와 결합하면, 이것은 <a href="eveline-re9-foreshadowing-explained.html">Vivien Ashcroft 연결 이론</a> 시사. RE10이 여기서 직접 설정됨.</p>
</div>
</details>

<h2 id="rank4">#4: Leon Secret 엔딩, "Old Debt" (A-티어)</h2>
<p><strong>#4인 이유:</strong> 21년 정사에서 Ada Wong의 첫 화면 "고마워". "고마워, Leon. 모든 것에 대해" 대사는 Ada의 감정적 마스크가 마침내 떨어지는 것으로 포지셔닝. "Albert의 아들" 미스터리가 RE10 악당을 티저. 더 많은 전제조건 필요하므로 Grace Secret보다 아래.</p>

<p><strong>해금 노력:</strong> ★★★ 중상 (Grace True 세이브 + NG+ 스탠다드+ + Ada 비콘 + "샘플 보존")</p>
<p><strong>최적 대상:</strong> Leon/Ada 관계 팬 (수십 년 아크), Wesker 혈통 추종자</p>
<p><strong>트로피:</strong> "Old Debt" (골드)</p>

<details>
<summary>▶ 공개: 왜 "고마워"가 중요한가</summary>
<div>
<p>RE2 (1998), RE4 (2004), RE6 (2013), Damnation (2009) 전반에 걸쳐, Ada는 화면에서 Leon에게 결코 "고마워"라고 말한 적 없음. 캡콤 디렉터는 대사가 17번 재작성됐다고 확인. 이것은 21년 Leon-Ada 아크가 마침내 움직이는 것. <a href="ada-wong-re9-reversal-arc.html">Ada Wong RE9 반전 분석</a> 참조.</p>
</div>
</details>

<h2 id="rank5">#5: Grace Insanity 엔딩, "The Cycle Continues" (B-티어)</h2>
<p><strong>#5인 이유:</strong> 헬리콥터가 알 수 없는 저격수에 의해 격추될 때까지 True 엔딩과 동일. Mia 생존, Grace 사망. 검은 장갑 손이 그녀의 시체를 들어올림. 이것은 캡콤이 Grace의 취약성과 프랜차이즈 불안정성에 대해 논평하는 것 — 그러나 만족스럽기보다 냉소적으로 다가옴.</p>

<p><strong>해금 노력:</strong> ★★★★ 어려움 (인새니티 + NG+ 캐리 무기 없음 + "샘플 파괴")</p>
<p><strong>최적 대상:</strong> 컴플리셔니스트, 가장 어두운 해석 원하는 사람</p>
<p><strong>트로피:</strong> "The Cycle Continues" (골드)</p>

<h2 id="rank6">#6: Leon Insanity 엔딩. "Quarantined" (B-티어)</h2>
<p><strong>#6인 이유:</strong> Leon이 탈출 중 감염, 혼자 격리 구역으로 걸어들어감. Helena에게 보내지 않은 "미안해" 문자는 감정적으로 무겁지만 Leon의 더 넓은 아크와 연결 안 됨. 의미 있는 캐릭터 해결이 아닌 비극적 대안 타임라인.</p>

<p><strong>해금 노력:</strong> ★★★★ 어려움 (인새니티 + 탈출 중 피격 + "샘플 보존")</p>
<p><strong>최적 대상:</strong> 컴플리셔니스트, "모두 죽음" 대안 타임라인 팬</p>
<p><strong>트로피:</strong> "Quarantined" (골드)</p>

<h2 id="rank7">#7: Grace Bad 엔딩. "Curiosity Killed the Hunter" (C-티어)</h2>
<p><strong>#7인 이유:</strong> Grace가 연구를 위해 Plant 43 샘플 보존 시도, 곰팡이가 그녀 흡수. 마지막 컷: Grace의 얼굴이 부분적으로 보이는 Plant 43. 이 엔딩은 잘못된 최종 선택을 처벌하고 트로피 제공하기 위해 존재. 주제적으로 작동하지만 지속적인 내러티브 가치 없음.</p>

<p><strong>해금 노력:</strong> ★ 쉬움 (어떤 난이도 + "샘플 보존")</p>
<p><strong>최적 대상:</strong> 트로피 컴플리셔니스트만</p>
<p><strong>트로피:</strong> "Curiosity Killed the Hunter" (실버)</p>

<h2 id="rank8">#8: Leon Bad 엔딩 — "The Hero's End" (C-티어)</h2>
<p><strong>#8인 이유:</strong> Leon이 모든 민간인을 구하지 않으면 떠나기 거부, 자폭 중인 ARK로 다시 뛰어들어감. "Helena에게 전해줘... 그녀 잘못이 아니라고" 대사는 가슴 아프지만 엔딩이 내러티브적으로 평평하게 느껴짐, 불가능한 구조를 위해 Leon이 죽는 것은 더 넓은 목적에 봉사하지 않음. Leon의 아크가 여기서 얻지 못하는 의미 있는 해결을 필요로 하므로 가장 낮은 랭킹.</p>

<p><strong>해금 노력:</strong> ★ 쉬움 (어떤 난이도 + Grace True 없이 "샘플 파괴")</p>
<p><strong>최적 대상:</strong> 트로피 컴플리셔니스트만</p>
<p><strong>트로피:</strong> "The Hero's End" (실버)</p>

<h2 id="common-mistakes">엔딩 추적 시 흔한 실수</h2>

<ol>
 <li><strong>Grace True 없이 Leon True 시도.</strong> Leon True는 Grace True 세이브 플래그로 게이팅. Grace True가 설정 안 됐다면 선택과 관계없이 Leon Bad 얻음.</li>
 <li><strong>NG+ 인새니티에서 Chunk 암호화 파일 놓침.</strong> Chunk 시체에 있음; NG+ 인새니티에서만 스폰, 일반 NG+ 아님. 많은 트로피 헌터가 여기서 플레이 낭비.</li>
 <li><strong>Grace Insanity 시도에서 NG+ 캐리 무기 사용.</strong> 게임이 추적. 캐리 무기 하나라도 사용하면 "The Cycle Continues" 자격 박탈. 대신 재시작.</li>
 <li><strong>기본값이 무엇인지 모르고 최종 선택 QTE 타임아웃.</strong> 기본값 = "샘플 파괴" = Grace True / Leon Bad. Grace Bad / Leon True를 원했다면 "보존" 적극 선택 필수.</li>
 <li><strong>Chunk 라운드 2 전에 저장 안 함.</strong> 자동 저장이 컷씬 중간. 수동 저장 없이는 엔딩 재시도 불가능. 항상 최종 보스 전에 USB나 클라우드로 백업.</li>
</ol>

<h2 id="faq">FAQ — 엔딩 랭킹</h2>

<p><strong>Q: 이 랭킹이 주관적인가요?</strong><br>
A: 부분적으로. 방법론은 5개 차원으로 점수 매기지만, 가중치와 개별 판단은 단일 관점 반영. Lore 팬은 상위 4개에 동의할 가능성; 트로피 헌터는 완료 가치 때문에 Bad 엔딩을 더 높게 랭킹할 수 있음.</p>

<p><strong>Q: 캡콤은 한 엔딩을 정사로 간주하나요?</strong><br>
A: 디렉터 Koshi Nakanishi는 Grace True + Leon True (둘 다 플레이)가 의도된 정사 경로임을 확인. 다음 게임의 공개 트레일러에 Mia Winters (Grace True)와 BSAA 전술 장비 입은 Leon (Leon True) 둘 다 등장.</p>

<p><strong>Q: 낮은 랭킹 엔딩을 해금하는 대신 YouTube에서 봐야 하나요?</strong><br>
A: Bad 엔딩 (#7, #8), 네, YouTube가 시간 절약. Insanity 엔딩 (#5, #6), 자신이 플레이할 가치 있음, 난이도가 엔딩에 감정적 무게 주기 때문. Secret 엔딩 (#3, #4) — 확실히 플레이; 트로피 진행과 프랜차이즈 lore 포함.</p>

<p><strong>Q: 방법론에 동의하지 않으면 랭킹이 바뀌나요?</strong><br>
A: 자신만의 랭킹을 만들기 위해 위의 5-차원 프레임워크 사용. 대안 랭킹에 관심 있음. re9uiem@re9guide.it.com로 이메일.</p>

<p><strong>Q: 잘려나간 엔딩이나 숨겨진 9번째 엔딩이 있나요?</strong><br>
A: 데이터마이닝된 파일 기준, 캡콤은 초기 제작 동안 11개 엔딩 개발 후 3개 잘랐음. 최종 게임의 8개 모두 정사 시나리오로 확인. 숨겨진 9번째 엔딩 없음.</p>

<p><strong>Q: 이 랭킹은 다른 RE 게임과 어떻게 비교되나요?</strong><br>
A: RE9의 True 엔딩 (#1, #2)은 내러티브 무게로 거의 모든 RE2/RE4/RE7 엔딩보다 위. Bad 엔딩 (#7, #8)은 RE9의 가장 약함. 대부분 다른 RE 게임은 더 의미 있는 "잘못된 선택" 엔딩 보유.</p>

<p><strong>Q: 미래 DLC가 새 엔딩을 추가할까요?</strong><br>
A: 캡콤은 확인하지 않음, 그러나 소문된 2026년 말 DLC는 Mercenaries 모드를 포함할 수 있음 (새 엔딩 아님). 엔딩을 추가하는 스토리 DLC는 프랜차이즈에 이례적.</p>
