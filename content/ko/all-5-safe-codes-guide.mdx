<h2 id="safe1">금고 #1 — Bar &amp; Lounge (카페 금고) // 케어 센터 서 윙</h2>
<p>대부분의 플레이어가 처음 만나는 금고. 바 카운터 뒤에 숨겨져 있어 놓치기 쉽습니다.</p>
<div>
<div>// 금고 #1, Bar &amp; Lounge (카페 금고) Safe</div>
<p>Normal: <strong>L10 · R80 · L30</strong></p>
<p>Insanity: <strong>L40 · R70 · L20</strong></p>
<p><strong>위치:</strong> 바 카운터 뒤, 꽃 장식 오른쪽. Bar &amp; Lounge의 좀비를 먼저 정리하세요. 다이얼 입력을 방해할 수 있습니다.</p>
<p><strong>코드 단서:</strong> 서 윙 사무실 책상 근처 클립보드에서 발견.</p>
<p><strong>보상:</strong> 앤틱 코인 3개 (Parlor에서 사용)</p>
</div>

<h2 id="safe2">금고 #2 — Examination Room // 케어 센터 동 윙 1층</h2>
<p>동 윙 1층 Examination Room. 코너 캐비닛에 자리잡고 있어 지나치기 쉽습니다.</p>
<div>
<div>// 금고 #2; Examination Room Safe</div>
<p>Normal: <strong>R30 · L10 · R50</strong></p>
<p>Insanity: <strong>R60 · L20 · R40</strong></p>
<p><strong>위치:</strong> Examination Room의 먼 코너, 바닥의 금속 캐비닛 안. 캐비닛에 눈에 보이는 긁힌 자국.</p>
<p><strong>코드 단서:</strong> 동 윙 2층, 거대한 뚱뚱한 좀비(청크)를 만나는 같은 방. 의료 차트를 보세요.</p>
<p><strong>보상:</strong> 앤틱 코인 3개</p>
</div>

<h2 id="safe3">금고 #3. Furnace Room // 케어 센터 지하</h2>
<p>접근이 가장 어려운 금고 중 하나 — 동 윙 2층의 첫 번째 Quartz Puzzle Box를 풀어서 얻는 Level 1 Wristband 필요. 이 퍼즐을 완료하기 전에 이 금고에 도달하려 하지 마세요.</p>
<div>
<div>// 금고 #3, Furnace Room Safe (Basement)</div>
<p>Normal: <strong>L60 · R20 · L40</strong></p>
<p>Insanity: <strong>L30 · R50 · L70</strong></p>
<p><strong>위치:</strong> 케어 센터 지하의 Furnace Room 안. 지하 보안 문 접근을 위해 Level 1 ID Wristband 필요.</p>
<p><strong>코드 단서:</strong> 정비 일정 근처 Boiler Room에서 파일 발견.</p>
<p><strong>보상:</strong> 샷건 셸 10발 + 앤틱 코인 2개</p>
</div>

<div>💡 <strong>보너스 팁:</strong> 지하의 Workshop에는 별도의 금고(R60·L40·R80)가 있으며 <strong>Raccoon Roundup 지도</strong>를 포함합니다. 이 지도는 모든 케어 센터 미스터 라쿤 메모리엄을 지도에 강조 표시합니다. 100% 런에 강력히 권장.</div>

<h2 id="safe4">금고 #4; Sterilization Chamber (실험실 금고) // 케어 센터 멸균</h2>
<p>이 금고는 케어 센터의 중반 최고 보상 일부를 포함하며, 청크와 셰프 같은 미니 보스를 한 방에 처치하는 헤모리틱 인젝터도 포함됩니다.</p>
<div>
<div>// 금고 #4 — Sterilization Chamber (실험실 금고) Safe</div>
<p>Normal: <strong>R70 · L50 · R10</strong></p>
<p>Insanity: <strong>R20 · L80 · R60</strong></p>
<p><strong>위치:</strong> Bioweapon Repository 근처 안방. 금고는 나무 상자 뒤에 부분적으로 숨겨져 있음. 옆으로 밀거나 쏴서 분해하세요.</p>
<p><strong>코드 단서:</strong> Sterilization Chamber 청정 구역 안의 연구 노트에서.</p>
<p><strong>보상:</strong> 헤모리틱 인젝터 2개 + 앤틱 코인 1개</p>
</div>

<h2 id="safe5">금고 #5. Monitor Control Room (관찰실 금고) // 라쿤 시티</h2>
<p>최종 금고는 라쿤 시티에 있으며, 케어 센터에서 멀리 떨어져 있습니다. <strong>매우 미스 가능</strong> — Bioweapon Repository 시퀀스를 지나가면 여기로 돌아올 수 없습니다.</p>
<div>
<div>// 금고 #5. 최종 금고. Monitor Control Room Safe</div>
<p>Normal: <strong>L90 · R30 · L60</strong></p>
<p>Insanity: <strong>L20 · R40 · L80</strong></p>
<p><strong>위치:</strong> Bioweapon Repository 05, 가장 오른쪽 열린 컨테이너. "Monitor Control"이라고 표시된 금속 락커를 찾으세요.</p>
<p><strong>코드 단서:</strong> Records Room의 보안 메모에서 발견.</p>
<p><strong>보상:</strong> 레퀴엠 탄환 5발 + 고급 약초</p>
</div>

<h2 id="tips">팁, 트릭, 트로피 보상</h2>

<h3>// 금고 코드 입력 방법</h3>
<p>금고에 접근하여 상호작용. 다이얼 퍼즐 화면이 나타납니다. 이 입력 사용:</p>
<ul>
<li><strong>L (왼쪽)</strong> = 다이얼을 시계방향으로 회전</li>
<li><strong>R (오른쪽)</strong> = 다이얼을 반시계방향으로 회전</li>
<li>나열된 숫자에 정확히 멈춤 — 지나치면 작동 안 함, 나가서 다시 시도</li>
<li>세 숫자 입력 후 Check / Confirm 버튼 누름</li>
</ul>

<h3>// 왜 그레이스만 금고를 열 수 있나</h3>
<p>금고 크래킹은 Blood Crafting처럼 그레이스 애쉬크로프트의 캐릭터 전용 능력 중 하나입니다. 레온은 금고와 전혀 상호작용 불가; 프롬프트조차 보이지 않습니다. 레온으로 플레이하면서 금고를 열려 하면, 스토리에서 그레이스 섹션으로 다시 전환하거나 Grace 전용 세이브를 사용하세요.</p>

<h3>// "No Safe Is Safe" 트로피</h3>
<p>5개 금고를 모두 여는 것(난이도 무관)이 "No Safe Is Safe" Silver 트로피를 해금합니다. 이 가이드를 손에 들고 있으면 게임에서 더 쉬운 수집품 기반 트로피 중 하나입니다.</p>

<h3>// 앤틱 코인으로 무엇을 할까</h3>
<p>금고에서 얻은 앤틱 코인은 케어 센터 1층의 Parlor에서 사용. 보상에는 일회용 부적, 자원 패키지, 그리고 (충분히 얻으면) Insanity-전용 Infinite Ammo 잠금 해제가 포함됩니다.</p>

<h2 id="speedrun">스피드런 전략: 코드 외우기, 단서 건너뛰기</h2>
<p>4시간 미만 스피드런을 노린다면, <strong>코드를 외우는 것</strong>이 결정적입니다. 각 금고의 단서를 찾는 데 평균 3-5분이 걸리며, 이 가이드의 코드를 사용하면 직접 코드 입력이 60-90초입니다.</p>
<ul>
<li>스피드런 런: 5개 금고 모두 직접 입력 = ~7-8분 절약</li>
<li>금고 #3과 #4는 케어 센터의 우회. 코어 스토리에 영향 없음</li>
<li>금고 #5(라쿤 시티)는 가장 가치 있는 보상 — 우선순위</li>
</ul>
<h2 id="faq">자주 묻는 질문</h2>

<div>

<details open>
<summary>RE9 카페 금고 코드는 어디에 있나요?<span>+</span></summary>
<p>"카페 금고"라고 부르는 것은 케어 센터 서윙의 <strong>Bar &amp; Lounge 금고 (Safe #1)</strong>입니다. 게임 내 정식 명칭은 "Bar &amp; Lounge"이지만, 라운지 구조 때문에 일부 커뮤니티에서 카페로 부릅니다. 코드는 스탠다드에서 <strong>L10 · R80 · L30</strong>, 인새니티에서 <strong>L40 · R70 · L20</strong>입니다.</p>
</details>

<details>
<summary>RE9 실험실(Lab) 금고 코드는?<span>+</span></summary>
<p>"실험실 금고"는 케어 센터 지하 실험실의 <strong>Sterilization Chamber 금고 (Safe #4)</strong>입니다. 챔버는 게임 내 정식 명칭이지만 플레이어들은 전체 실험실 구역을 "Lab"이라고 부릅니다.</p>
</details>

<details>
<summary>RE9에 관찰실 금고가 있나요?<span>+</span></summary>
<p>네, "관찰실(Observation Room) 금고"는 라쿤 시티 구역의 <strong>Monitor Control Room 금고 (Safe #5)</strong>를 부르는 별칭입니다. CCTV 모니터 벽 때문에 정식 명칭은 Monitor Control Room이지만, 관찰 데크 구조 때문에 Observation Room으로 불립니다.</p>
</details>

<details>
<summary>RE9에 금고가 총 몇 개 있나요?<span>+</span></summary>
<p>총 <strong>5개</strong>입니다: Bar &amp; Lounge (카페), 검사실, 보일러실, Sterilization Chamber (실험실), Monitor Control Room (관찰실). 5개 모두 열면 <strong>"Safe Master" 트로피</strong> 획득.</p>
</details>

<details>
<summary>금고 코드는 플레이마다 바뀌나요?<span>+</span></summary>
<p>아니요; 난이도별로 <strong>고정</strong>입니다. 스탠다드 코드는 모든 스탠다드 런에서 동일, 인새니티 코드는 모든 인새니티 런에서 동일.</p>
</details>

<details>
<summary>금고를 영원히 놓칠 수 있나요?<span>+</span></summary>
<p>네 — 금고 #1-#4은 모두 케어 센터에 있으며, 코트야드/워터 트리트먼트로 떠나면 같은 런에서 <strong>돌아갈 수 없습니다</strong>. <a href="re9-missables-complete-guide.html">RE9 놓치기 쉬운 항목 가이드</a> 참조.</p>
</details>
