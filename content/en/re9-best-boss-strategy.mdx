<p><strong>Short answer:</strong> Universal strategy: <strong>Requiem Revolver headshots + W870 Police shotgun close-range + Knife parries</strong>. Equip <strong>Crafter's Touch + Iron Skin + Phoenix Down</strong> charms. Specific boss strategies below. All 10 bosses have detailed guides linked.</p>
<h2>The universal boss strategy</h2>
<ol>
 <li><strong>Pre-fight prep:</strong> Stock 8-12 Requiem Revolver rounds, 4-6 W870 Police shotgun shells, 3 First Aids minimum</li>
 <li><strong>Charm loadout:</strong> Crafter's Touch + Iron Skin + Phoenix Down</li>
 <li><strong>Opening:</strong> Requiem Revolver to head until first phase transition</li>
 <li><strong>Mid-fight:</strong> W870 Police shotgun close-range during low-mobility phases</li>
 <li><strong>Closing:</strong> Knife parry for execution / cutscene triggers</li>
</ol>

<h2>Boss-specific best strategies</h2>

<h3>Victor Gideon (Final Boss)</h3>
<p><strong>Best strategy:</strong> Requiem Revolver headshots Phase 1-2 (4 headshots), W870 Police shotgun close-range Phase 3, knife parry for QTE finale (0.4-second window).</p>
<p><a href="how-to-beat-victor-gideon.html">Full Victor Gideon guide →</a></p>

<h3>Mr. X / Super Tyrant</h3>
<p><strong>Best strategy:</strong> Circle-strafe, never back-pedal. Use RPD pillars for 90° cover. W870 Police shotgun close-range during stagger. 6 Requiem Revolver rounds total.</p>
<p><a href="how-to-beat-mr-x-super-tyrant.html">Full Mr. X guide →</a></p>

<h3>Chunk (Final Encounter)</h3>
<p><strong>Best strategy:</strong> AR full-auto on arms (break both for execution opening). W870 Police shotgun for Phase 2 mutation. Requiem Revolver to head for Phase 3 truth reveal.</p>
<p><a href="how-to-beat-chunk.html">Full Chunk guide →</a></p>

<h3>Plant 43</h3>
<p><strong>Best strategy:</strong> AR full-auto on 3 root clusters (15 rounds each). Avoid spore cloud — run to room edge. Flame rounds skip Phase 2.</p>
<p><a href="how-to-beat-plant-43.html">Full Plant 43 guide →</a></p>

<h3>Titan Spinner</h3>
<p><strong>Best strategy:</strong> Sidestep last-second (0.3s before charge). Climb east fire escape Phase 2. Grenades knock down (15% HP each). Free-fall finisher.</p>
<p><a href="how-to-beat-titan-spinner.html">Full Titan Spinner guide →</a></p>

<h3>The Girl</h3>
<p><strong>Best strategy:</strong> Bloodhound charm to track. Stealth approach (must be unalerted). Back-blast W870 Police shotgun for first hit. Drowning execution = True Ending.</p>
<p><a href="how-to-beat-the-girl.html">Full The Girl guide →</a></p>

<h3>The Commander</h3>
<p><strong>Best strategy:</strong> Wait for shield drop (every 4s) → Requiem Revolver headshot. W870 Police shotgun for twin ad spawns. Phoenix Down activates if you die.</p>
<p><a href="how-to-beat-the-commander.html">Full Commander guide →</a></p>

<h3>Blister Borne</h3>
<p><strong>Best strategy:</strong> 5m distance maintain. S&S M232 S&S M232 handgun whittle. Sprint behind cover during pop (360° AOE). Knife-only kill = Predator charm.</p>
<p><a href="how-to-beat-blister-borne.html">Full Blister Borne guide →</a></p>

<h3>The Chef (Optional)</h3>
<p><strong>Best strategy:</strong> Don't pick up meat lures. Predictable knife arcs, sidestep + reply fire. Knife counter-stab does double damage.</p>
<p><a href="how-to-beat-the-chef.html">Full The Chef guide →</a></p>

<h2>Insanity-specific boss tactics</h2>
<ul>
 <li><strong>Phoenix Down is mandatory</strong>, saves 1 death per chapter</li>
 <li><strong>Pre-stock 6+ First Aids</strong> for Chapter 8 finale (4 bosses back-to-back)</li>
 <li><strong>Requiem Revolver headshots only</strong> — Insanity scales body damage minus</li>
 <li><strong>Save grenades for vertical Spinner</strong>. best DPS option</li>
 <li><strong>Crafter's Touch refills mid-chamber</strong>, never run dry</li>
</ul>
