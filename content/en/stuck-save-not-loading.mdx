<h1>Stuck: save not loading — corruption recovery</h1>
<p>Save state corruption is rare but happens. Specific recovery strategies for different corruption types.</p>

<blockquote>Save corruption is real but recoverable in most cases. Cloud backup is essential.</blockquote>

<h2>If save fails to load</h2><p>Try alternate save slot (rotate through all 6). Try latest auto-save. Restart console + retry.</p><h2>If all save slots fail</h2><p>Cloud backup recovery (PS5 / Xbox). Steam cloud recovery. Possibly chapter restart from last known good state.</p><h2>Prevention</h2><p>Enable cloud backup. Rotate through 6 save slots. Don't quit during cutscenes or transitions.</p><h2>Reporting bugs</h2><p>Capcom Korea support has been responsive to Korean RE9 save bug reports. International reporting through official Capcom channels.</p><h2>My take</h2><p>Save corruption is rare but devastating. Backup obsessively. Cloud backup is non-negotiable for Insanity playthroughs.</p>
