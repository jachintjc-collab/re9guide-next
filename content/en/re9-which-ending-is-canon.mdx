<p><strong>Short answer:</strong> The <strong>Grace True Ending</strong> is the official canon. The <strong>Leon Insanity Secret Ending</strong> is alt-canon (sets up RE10). All other endings are non-canonical. Director Nakanishi confirmed this in the May 2026 Famitsu interview.</p>
<h2>The canon ending: Grace True Ending</h2>
<p>Achieved by: surviving Victor Gideon's QTE finale with the knife parry (0.4-second window) on Standard difficulty or higher.</p>
<p><strong>What happens:</strong> Grace escapes ARK Facility. Mia Winters arrives in EW-2017 helicopter. Vivien-Eveline connection cinematic plays. Grace returns to Seoul carrying Eveline's Locket. Sets up Grace as RE10 protagonist.</p>

<h2>The alt-canon: Leon Insanity Secret Ending</h2>
<p>Achieved by: completing NG+ Insanity as Leon + missing the Victor Gideon QTE parry intentionally.</p>
<p><strong>What happens:</strong> Leon survives with severe injuries. Ada Wong beacon plays — she's tracking him. Sets up DLC and/or RE10 reunion.</p>

<h2>Non-canon endings</h2>
<ul>
 <li>Grace Bad Ending. Grace fails QTE on Casual</li>
 <li>Leon True Ending, succeeds but doesn't trigger Ada beacon</li>
 <li>Bad Endings (Grace + Leon) — story ends with character death</li>
 <li>Insanity Standard Endings. same content as Standard with bonus codex</li>
</ul>

<h2>How to confirm canon: official sources</h2>
<ul>
 <li><strong>Famitsu interview (May 2026):</strong> Director Nakanishi states Grace True Ending is canon</li>
 <li><strong>RE9 official artbook:</strong> Includes only Grace True Ending art</li>
 <li><strong>Capcom community posts:</strong> Reference "Grace's journey continues", implies survival</li>
</ul>
