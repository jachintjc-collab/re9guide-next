<h1>Leon True Ending — alternative true ending route via Leon dialogue</h1>
<p>An alternative true ending unlock route through Leon-cooperation dialogue choices. Different from Grace True Ending — same trophy hint but different content. Estimated 7% of players find this without a guide.</p>

<blockquote>Leon True Ending is one of RE9's hidden depth rewards. Most players don't realize a second true ending route exists.</blockquote>

<h2>What makes this different from Grace True Ending</h2>
<p>Grace True Ending: Grace acts independently, publishes findings, Sherry Birkin allies post-credits.</p>
<p>Leon True Ending: Grace cooperates extensively with Leon, BSAA institutional faction war happens with Leon as primary opposition figure rather than Sherry.</p>
<p>Same post-credits Korean BSAA commander scene but with Leon's name on her briefing folder instead of Sherry's.</p>

<h2>The unlock conditions</h2>
<p>Choose Leon-cooperation dialogue options consistently across chapters 4, 6, 7, and 8 (not just one). The choices position Grace as Leon's partner rather than independent investigator.</p>
<p>Specifically: chapter 4 "Yes, send backup" instead of "I'll handle it." Chapter 6 "Let me check with Leon" instead of "I'm proceeding." Chapter 7 "Trust Leon's judgment" instead of "My call." Chapter 8 final dialogue: "Leon will know what to do" instead of "I'll handle this."</p>

<h2>Why most players miss this</h2>
<p>The default game progression nudges toward Grace-independent choices. The dialogue UI doesn't visually distinguish Leon-cooperation paths. Players who instinctively choose "Grace acts alone" framing miss the entire alternative route.</p>

<h2>The narrative impact</h2>
<p>Leon True Ending establishes Leon as RE10's institutional opposition lead rather than Sherry. The franchise direction shifts depending on which true ending the player gets to first.</p>
<p>Capcom hasn't confirmed which true ending is canonical for RE10. Both will likely be acknowledged.</p>

<h2>Trophy timing</h2>
<p>Trophy fires at post-credits scene end. Both true endings produce the same trophy unlock — you don't need both for the trophy. Pursue whichever fits your dialogue preferences.</p>

<h2>Korean community on the dual true endings</h2>
<p>Korean Naver lore community has documented both true ending paths with comparison videos. Korean reception was split — some readers preferred Sherry as ally (Grace True), others preferred Leon (Leon True).</p>

<h2>My take</h2>
<p>Leon True Ending is the hidden depth reward for engaged dialogue players. If you naturally lean cooperative in dialogue choices, you may end up with Leon True Ending without knowing the Grace True route existed. Both are valid.</p>
