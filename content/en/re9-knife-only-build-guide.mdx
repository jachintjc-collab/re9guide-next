<p><strong>The Knife-Only Build is the franchise&rsquo;s hardest player-imposed challenge.</strong> Mortal Edge is your only weapon. No firearms. Built on parry mastery. Knife Edge charm (+30% damage) is mandatory. Insanity NG+ achievable but requires 100% parry precision.</p>

<h2>Complete Loadout</h2>
<div>
 <table>
 <tr><th>Slot</th><th>Pick</th></tr>
 <tr><td>Primary weapon</td><td>Mortal Edge (only)</td></tr>
<tr><td>Secondary</td><td>None, no firearms allowed</td></tr>
<tr><td>Charm 1</td><td>Knife Edge (+30% damage)</td></tr>
<tr><td>Charm 2</td><td>Iron Skin (close combat defense)</td></tr>
<tr><td>Charm 3</td><td>Phoenix Down (revive once)</td></tr>
<tr><td>NG+ Charm 4</td><td>Survivor&rsquo;s Will (herbs +50%)</td></tr>
<tr><td>Item</td><td>Maximum herbs (no ammo carried)</td></tr>
<tr><td>Best chapter</td><td>Knife-only Blister Borne kill (Chapter 2, unlocks Predator)</td></tr>
 </table>
</div>

<h2>Why this build works</h2>
<p>This build is balanced for its specific playstyle. Each component serves a clear purpose — weapons handle damage output, charms enhance survivability or efficiency, and items provide specialized utility.</p>

<h2>Difficulty recommendations</h2>
<ul>
 <li><strong>Casual:</strong> Easy mode, use this build to learn mechanics</li>
 <li><strong>Standard:</strong> Intended difficulty; this build works well</li>
 <li><strong>Insanity:</strong> Recommended for skilled players who have completed Standard</li>
 <li><strong>Insanity NG+:</strong> Optimal. Infinite Magnum and Phoenix Down unlock</li>
</ul>

<h2>Recommended playthrough order</h2>
<ol>
 <li>Complete Standard NG first (12-15 hours)</li>
 <li>Start NG+ with this build equipped</li>
 <li>Move to Insanity NG+ when comfortable (15-18 hours)</li>
 <li>Optional: speedrun for Platinum trophy</li>
</ol>
