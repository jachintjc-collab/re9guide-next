<h2>Frame-Reading Method</h2>
<p>Insanity dodges that "feel impossible" usually have a 4-frame window. To recognize these in real time:</p>
<ul>
 <li><strong>Watch the boss elbow/shoulder pivot</strong>, not the weapon. Limb wind-up starts 4-6f before the strike.</li>
 <li><strong>Audio cue layering:</strong> low-frequency growl precedes phase 4 attacks by exactly 0.2s. Use this for blind-side dodges.</li>
 <li><strong>Reduce input latency:</strong> wired DualSense + 60Hz display = 16ms input lag. Wireless adds 30-50ms — never recommended on Insanity.</li>
</ul>
