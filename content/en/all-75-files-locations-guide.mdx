<h2 id="how">How Files work</h2>

<p>Files are document collectibles scattered across the game world. Some give you puzzle clues (like safe code hints), others fill in lore. They're tracked in the <strong>Files tab</strong> next to your inventory; but the in-game order doesn't always match the chronological order you find them in.</p>

<ul>
 <li><strong>Reading is mandatory</strong> — picking a File up isn't enough; you must open and view it</li>
 <li><strong>Both Grace and Leon collect Files</strong>; the total of 75 spans both characters</li>
 <li><strong>Some Files are story-automatic</strong>, you can't miss them, they trigger via cutscene</li>
 <li><strong>Most Files are environmental</strong> — you have to actively look around rooms</li>
 <li><strong>Reward: Case Closed Silver trophy + 3,000 CP</strong> from the Challenge menu</li>
</ul>

<h2 id="tips">Tips and missable warnings</h2>

<h3>// TOP MISSABLE FILES</h3>

<p>Based on community feedback, these are the most commonly missed Files:</p>

<ul>
 <li><strong>File #14 (Director's Notes)</strong>; Requires the Pencil from a separate room. Skip this and you skip the safe code clue too.</li>
 <li><strong>File #32 (Pantry Inventory)</strong> — Locked behind The Chef boss fight. Skipping the Chef means losing this file forever.</li>
 <li><strong>Files #39-42 (Marie's Drawing + Subject 17 Reports)</strong>, Critical for NG+ Final Puzzle lore. Don't rush past Collection &amp; Testing.</li>
 <li><strong>File #57 (RPD Library Note)</strong>; Easy to miss during the Mr. X chase. Return to the library after defeating Super Tyrant.</li>
 <li><strong>Files #70-73 (ARK Access Logs &amp; Safes)</strong> — All locked behind the final ARK sequence. Once you trigger the Victor cutscene, they're gone.</li>
</ul>

<h3>// EFFICIENT COLLECTION ROUTE</h3>

<p>For first-time 100% runs:</p>

<ol>
 <li><strong>Save before each area transition</strong>, use multiple slots</li>
 <li><strong>Clean Wrenwood completely</strong> before the hotel cutscene</li>
 <li><strong>Care Center: do 1F → 2F → Basement</strong> in that order, never the reverse</li>
 <li><strong>Defeat Chef + Chunk #1</strong> as Grace for missable files</li>
 <li><strong>Return to RPD Library</strong> after Mr. X is defeated for File #57</li>
 <li><strong>ARK: clean all files BEFORE</strong> entering the final Archive console</li>
</ol>

<h3>// IF YOU MISS A FILE</h3>

<p>If you progress past a point of no return without all 75 files, your options are:</p>

<ul>
 <li><strong>Reload a manual save</strong> from before the transition; best option if you saved often</li>
 <li><strong>Continue to credits and start NG+</strong> — files reset, but you keep your weapon unlocks and Infinite Ammo</li>
 <li><strong>Use Insanity speedrun strategy</strong>, knowing all locations from this guide, a complete run takes ~4 hours with Infinite Requiem equipped</li>
</ul>

<p>The Files menu shows you exactly which file numbers you're missing, making it easy to plan a targeted second playthrough.</p>
